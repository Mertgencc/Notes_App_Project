### 🧾 Notes App

A simple Notes application built with HTML, CSS, and JavaScript. This project allows users to create and delete notes dynamically in the browser using basic DOM manipulation.

---

#### 🚀 Features
- Add new notes
- Delete existing notes
- Simple and clean UI
- Instant updates without page refresh

---

#### 🛠️ Technologies Used
- HTML
- CSS
- JavaScript (DOM Manipulation)

---

#### 📌 Purpose
This project was created to practice JavaScript DOM manipulation and improve understanding of dynamic UI updates.
